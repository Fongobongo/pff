# Project Guidelines

## Language

- **English only** across the entire project:
  - documentation
  - code
  - comments
  - commit messages (preferred)
  - issue/PR text (when applicable)

## Resources / Cost

- Use **free-tier resources only** (hosting, database, APIs, etc.).
- Preferred database: **Supabase** (free tier).

## Sources & Data

- Blockchain analytics target network: **Base** (Ethereum L2).
- Player statistics must be sourced from the web and must include **all metrics used by the game to compute points**.

## Logs

- `docs/CONVERSATION_LOG.md` — curated decisions/requirements from chats (English).
- `CHANGELOG.md` — project changes over time (English).
